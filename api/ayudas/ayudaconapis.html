Guías de Ayudas y Errores

Solución de Problemas Comunes

Si tienes problemas al subir imágenes, revisa los siguientes errores frecuentes de la API:

  • Error: Faltan datos: Asegúrate de estar enviando el apiKey, usuarioID y el archivo de imagen en el FormData.
  • Error: API Key no válida: Verifica que la clave HI-XXXX coincida con la que aparece en tu sección "Conseguir API".
  • Error en almacenamiento: Esto ocurre cuando el Token de GitHub ha caducado o el repositorio no tiene permisos de escritura.
  • Error de conexión: Revisa que la URL de destino sea https://hostingimage-bice.vercel.app/upload.php.

Consejos de Implementación

Para un rendimiento óptimo, te recomendamos:

  • Usar imágenes en formato JPG o PNG para reducir el peso.
  • No subir archivos mayores a 5MB para evitar cortes en la conexión.
  • Implementar la subida de forma asíncrona usando fetch o AJAX.
