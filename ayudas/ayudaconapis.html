Guía: Cómo conectar la API

Para integrar nuestro servicio de hosting en tu propia web o aplicación, utiliza el siguiente ejemplo base. Solo tienes que sustituir los valores de configuración por los tuyos.

Parámetros de Configuración

PARÁMETRO DESCRIPCIÓN
apiKey Tu clave secreta HI-XXXX (la encuentras en "Conseguir API").
usuarioID Tu identificador único de sistema.
path Ruta donde se guardará en el servidor (ej: /imagenes/cliente1/).

Código de ejemplo (HTML + JavaScript)

<!-- Copia este código en tu archivo HTML -->
<script>
  const formData = new FormData();
  formData.append('image', archivoInput.files[0]);
  formData.append('apiKey', 'TU_API_KEY');
  formData.append('usuarioID', 'TU_USUARIO_ID');
  formData.append('path', '/imagenes/mi-empresa/');

  fetch('https://hostingimage-bice.vercel.app/upload.php', {
      method: 'POST',
      body: formData
  })
  .then(res => res.json())
  .then(data => {
      if(data.success) {
          console.log("URL de la imagen: " + data.url);
          document.getElementById('mi-img').src = data.url;
      }
  });
</script>

¿Cómo funciona?

  1. El usuario selecciona una imagen en tu web.
  2. Tu código envía la imagen junto a tu API Key a nuestro servidor.
  3. Nosotros procesamos la subida y te devolvemos una URL directa.
  4. Esa URL es la que debes guardar en tu base de datos para mostrar la imagen siempre.
